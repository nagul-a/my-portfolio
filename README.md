# Nagul A — Portfolio

My personal portfolio site. Built it from scratch using plain HTML, CSS, and a bit of vanilla JS — no frameworks, no build tools, just open the file and it works.

**Live:** [nagul-a.github.io/nagul-portfolio](https://nagul-a.github.io/nagul-portfolio)

## What's inside
- Hero section with typing animation
- About, Skills, Projects, Contact sections
- Responsive on mobile
- Fade-in scroll animations

## Tech
HTML · CSS · JavaScript

## Run it locally
Clone the repo and open `index.html` in any browser. That's it.

```bash
git clone https://github.com/nagul-a/nagul-portfolio.git
cd nagul-portfolio
# open index.html
```

## Deploy (GitHub Pages)
1. Go to repo Settings → Pages
2. Source: `main` branch, `/ (root)`
3. Save — it'll be live in a minute
